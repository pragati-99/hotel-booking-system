\# 🏨 Hotel Booking System



A complete Hotel Booking Management System built with \*\*Spring Boot\*\* (Backend) and \*\*React + Vite\*\* (Frontend).



\## ✨ Features



\### 👤 Guest Features

\- View available rooms with images

\- Book rooms with online payment (Razorpay)

\- View booking history

\- Request refunds

\- Pre-booking

\- Contact support



\### 🛎️ Receptionist Features

\- Check-in/Check-out guests

\- Manage bookings

\- Manage customers \& drivers

\- Manage refunds \& pre-bookings



\### 👑 Admin Features

\- Full CRUD for Rooms, Employees, Customers, Drivers

\- Manage all bookings

\- Approve/Reject refunds

\- Manage pre-bookings

\- Reports \& analytics

\- Department management



\## 🛠️ Tech Stack



\### Backend

\- Java 17

\- Spring Boot 3.2.3

\- Spring Security with JWT

\- Hibernate/JPA

\- MySQL

\- Razorpay Payment Gateway



\### Frontend

\- React 18

\- Vite

\- Bootstrap 5

\- React Router DOM v6

\- Axios

\- React Hot Toast
